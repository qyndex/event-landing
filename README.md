# Event Landing

Event landing page with countdown timer, speaker bios, schedule, and registration form.

**Stack:** Astro | **Complexity:** moderate
**Tags:** `event`, `landing`, `astro`, `countdown`, `registration`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
